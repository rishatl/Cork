User Interface

Cork is made up of three main sections: the Sidebar, the Detail Area, and the Toolbar. Other elements, such as package installation sheets, may be shown over the main sections.

Sidebar

The sidebar shows a list of installed packages and taps, and a Search Field which you can use for finding packages.

See Sidebar for more information

Detail Area

The detail area is where you’ll spend most of your time. Depending on the context, it might show you the startup page or detailsa bout the item currently selected in the sidebar.

See Detail Area for more information.

Toolbar

The toolbar gives you access to actions that modify the packages you have installed, install new packages, and more.

See Toolbar for more information.
